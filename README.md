# Decentralized-Law-Chinese (去中心化法律 - 中文版)

欢迎来到 **《Decentralized Law》（去中心化法律）** 的开源中文翻译项目。

## 项目愿景
曾几何时，区块链技术承诺要重构世界的治理。带来全新的金融体系、全新形式的公司企业，乃至全新的国家。然而，这一切并未如愿发生。

为什么？因为统治世界的终究是法律，而非账本上的数字。如果无法在现实世界中影响和触及法律，这个行业便永远无法释放其全部潜力。

本项目旨在通过系统性的翻译与整理，将去中心化法律的前沿思想引入中文社区，探讨区块链如何与法律融合，并为二十一世纪的去中心化经济构建坚实的法理基础。

## 关于本书
本书不仅是对去中心化技术的分析，更是一场关于法律本质的探索。我们将带你穿越历史，重新审视法律的四大源头：自然法、民法、国际法与私法。书中深入探讨了智能合约、代币化、DeFi、DAO 以及去中心化司法等核心议题。

## 目录预览
* **第一部分：法律的四大源头**（自然法、民法、国际法、私法）
* **第二部分：区块链法律**（智能合约、代币化、DeFi、去中心化组织与司法）
* **第三部分：去中心化法律的未来**（自由节点、去中心化自治协会 DAA、法律的去中心化路径）
